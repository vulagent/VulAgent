# bsaeline

This directory contains various files and datasets related to our experimental evaluation. Below is an overview of each item:

- **BigSleep_baseline.csv**  
  Records the test results of using the BigSleep application with different large language models to detect discovered vulnerabilities.

- **PDGF.zip**  
  Contains twenty vulnerability samples tested using the PDGF fuzz testing tool, documented as images.

- **sf_data1.zip**  
  Includes twenty vulnerability samples tested with the Select Fuzz fuzz testing tool. The testing was interrupted due to uncontrollable circumstances; both images and status files are included.

- **sf_data2.zip**  
  Documents the continued testing using Select Fuzz, with results recorded in image format.

- **sf_data3.zip**  
  Documents the continued testing using Select Fuzz, with results recorded in status files.
